# Melbourne Property Analysis

## Did Remote Work Change Melbourne's Location Premium?

This project investigates whether the COVID-19 pandemic altered 
residential property price gradients in metropolitan Melbourne, 
Australia. Specifically, it examines whether suburbs farther from 
the CBD appreciated more during the pandemic than before.

## Key Findings

- Before the pandemic, distance and transport access explained 
26% of price variation across suburbs (R² = 0.26)
- During the pandemic, this pattern collapsed to just 1.7%, 
as appreciation became widespread and generalised
- In the post-pandemic period, the pattern partially recovered 
(R² = 0.18), but train access lost statistical significance

## Data

Victorian Property Sales Report (Victorian Government, 2013-2023)
393 metropolitan Melbourne suburbs

## Tools

Python, pandas, statsmodels, matplotlib

## Reference
